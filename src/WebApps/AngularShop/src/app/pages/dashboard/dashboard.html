Routing

All API calls are sent to relative paths. In local dev the Angular dev-server proxy forwards them to http://localhost:6004 (YARP gateway). In Docker the Angular container's nginx forwards them to http://yarpapigateway:8080. The browser always sees a same-origin call, so no per-service CORS is required.
